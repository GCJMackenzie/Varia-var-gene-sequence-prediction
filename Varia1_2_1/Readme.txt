Use by running command:
Varia.sh [Name_of_input.fasta] [Identity_score (1-100)]

Input file must be in the fasta format or Varia will not run
Identity score must be an integer from 1 to 100 or Varia will not run
Input file and identity score must both be present and in the correct order or Varia will not run

Output is stored in directory [name_of_input_file]-[identity_score]-Varia_Out
